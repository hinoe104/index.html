# Luminous Salon - Integrated Fortune Launcher

## 概要 (Overview)
Luminous Salon は、タロットカードとルノルマンカードを用いた、熟練者および愛好家のための占術実行ツールです。
精密なカード画像を通じて、視覚的にリーディングできます。

## 特徴 (Features)
- **タロット**: ライダー・ウェイト版（全78枚）に対応。
- **ルノルマン**: 1799年原典「Das Spiel der Hoffnung」のデザインを精密にデジタル化。
- **軽量・高速**: シンプルな HTML/CSS/JS で構成され、あらゆるデバイスで快適に動作します。

画像素材については、それぞれの元の権利状態に基づき、自由な活用を推奨します。
- **タロット画像**: パブリックドメイン (1909年版)。本プロジェクトでは [metabismuth/tarot-json](https://github.com/metabismuth/tarot-json) の成果物を活用しています。
